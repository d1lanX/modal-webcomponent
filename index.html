modal webcomponent

this component tries to ease the handling of modals with local html content, as well as
html responses from a server.

what does this mean?

it basically means you can set html as you would normally do, and later activate the modal:

attributes

  • size: (sm, md, lg, full)
  • position: (rt, lt, lb, rb, center)
  • header: string
  • button: boolean
